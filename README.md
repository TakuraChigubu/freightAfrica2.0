# FreightLink Africa 🚚

An AI-powered freight marketplace connecting freight brokers, shippers, transport companies, and fleet owners across Africa.

## 📋 Project Overview

FreightLink Africa is a production-grade enterprise SaaS platform designed to revolutionize the freight and logistics industry in Africa.

### Initial Market
- Zimbabwe ↔ South Africa
- Expandable architecture for pan-African growth

## 🏗️ Architecture

### Technology Stack

**Backend:** Node.js, Express 5, PostgreSQL, Redis, JWT, Bcrypt, Google OAuth 2.0
**Frontend:** React, Vite, TypeScript, React Router DOM, Axios

## 🚀 Getting Started

### Prerequisites
- Node.js >= 18.0.0
- PostgreSQL >= 13
- Redis >= 7
- Docker & Docker Compose (recommended)

### Installation

1. Clone the repository
2. Run docker-compose up -d
3. Install dependencies
4. Start development servers
